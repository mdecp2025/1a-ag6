Brython 是什麼？

Brython 是一個讓 Python 程式碼能直接在瀏覽器中執行的工具，透過將 Python 轉譯為 JavaScript 來運作。

在瀏覽器中配置 Brython，只需在 HTML 中引入 Brython 的 JS 檔案，並使用 type="text/python" 撰寫 Python 程式即可。

Brython 能操作 DOM、處理事件、與瀏覽器 API 互動，適合用於互動式網頁、教學與小型前端應用。
