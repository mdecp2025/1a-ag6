WH3

定義

git 是一套「分散式版本控制系統」，用來：

  • 追蹤檔案的修改歷史

  • 協助多人協作開發

  • 可以隨時回到過去版本

GitHub 是一個「git 版本庫的雲端平台」

gist 是 GitHub 提供的「小型 git 專案」

cmsimde 是一個：結合 git + GitHub + 網頁 CMS 的「課程/專案管理系統」

工具 角色 功能
git 核心工具 版本控制
GitHub 雲端平台 備份、協作
gist 輕量版 GitHub 小程式、筆記
cmsimde 整合系統 教學 + 模擬

為什麼要使用這些工具？

因為現代學習、工程與動態模擬需要：可追蹤、可回復、可協作、可展示，而這些工具正好分工完成這四件事。

WH7

Brython 是什麼？

Brython 是一個讓 Python 程式碼能直接在瀏覽器中執行的工具，透過將 Python 轉譯為 JavaScript 來運作。

在瀏覽器中配置 Brython，只需在 HTML 中引入 Brython 的 JS 檔案，並使用 type="text/python" 撰寫 Python 程式即可。

Brython 能操作 DOM、處理事件、與瀏覽器 API 互動，適合用於互動式網頁、教學與小型前端應用。
